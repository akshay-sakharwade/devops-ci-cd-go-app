# DevOps CI/CD Pipeline with Golang

## Overview
This project demonstrates a CI/CD pipeline using Jenkins and Docker with a Golang web application.

## Flow
1. Code pushed to GitHub
2. Jenkins triggers pipeline
3. Docker image is built
4. Container is deployed on port 8080

## Tools Used
- Golang
- Jenkins
- Docker
- GitHub

## Outcome
Automated deployment and reduced manual effort.
